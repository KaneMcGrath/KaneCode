# KaneCode
KaneCode is a small, personal IDE project for experimenting with new ideas for developing C# software and using local LLMs.

![alt text](https://github.com/KaneMcGrath/KaneCode/blob/master/Icons/Capture.png?raw=true "Screenshot")

KaneCode is currently capable of simple C# app development, but it is not yet recommended for any production use case.
There are still many bugs and incomplete features.

# Features
- Solution and Project loading
- AvalonEdit code editor with syntax highlighting
- integrated build and run
- AI chat with agent mode tool loop
  - read_file
  - write_file
  - edit_file
  - delete_file
  - rename_path
  - create_directory
  - delete_directory
  - list_files
  - search_files
  - run_build
  - get_diagnostics
- Teacher mode with tools for navigating and presenting the code base
  - presentation_new
  - presentation_add_slide
- Use any OpenAI api provider including locally ran Llama.cpp
- Integrated Git tools
